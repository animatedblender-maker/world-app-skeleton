# Mobile Packaging (Capacitor)

This folder packages the web build for iOS/Android using Capacitor.

## Quick start
1. Build the web app: `npm run build` from `apps/web`.
2. From this folder: `npm run cap:sync`
3. Open native projects:
   - `npm run cap:ios`
   - `npm run cap:android`

## Notes
- `webDir` points to `apps/web/dist/web`.
- Native folders (`ios`, `android`) are generated by Capacitor.
